# 8_GigsForGigs
Problem Statement: 
Creator/Gig Worker Onboarding & Profile Platform
Identified Actors: A gig professional, an individual client and a Manager for an organisation client.
Features for each of the Identified actors:
1) Gig professional: 
    a) Creation of profile (Role               selected:Gig professional):             Verification of the professional 
    b) Post a gig
    c) Manage service requests
    d) View Service Opportunities
2) Individual Client:
    a) Creation of profile
    b) Talent discovery and engagement
    c) Manage and monitor selected gig         professional
    d) Payment and review the gig              professional

3) Manager- Organisation Client
    a) Creation of profile
    b) Host tasks to hire gig                  professional
    c) Manage and monitor selected gig         professional
    d) Payment and review the gig              professional
